Project 1: Fire Fighting Robot

Project Overview

This model of Fire Extinguishing Robot aids to share out the burden of fire fighters in firefighting task. Our project aims to build a real time fire fighting robot which moves in a constant speed, identify the fire and then extinguish it with the help of pumping mechanism. The detection and extinguishing was done with the help basic hardware components attached with the robot. Firstly, IR Flame sensors are used for the detection of fire. Secondly, BO Motors and Rubber wheels are used to navigate the robot to reach the fireplace. Finally, the robot extinguishes the fire with the help of submersible water pump and servo motors.

Project Details

Components used:

• Arduino UNO

• Flame sensor

• Servo Motor (SG90)

• L293D motor Driver module

• Mini DC Submersible Pump

• BO Motors

Functionality of the project in few steps:

• Flame sensor detecting the fire

• Sending the signal to Arduino

• Analyzing the presence of fire

• Activation of water pump

• Putting off the fire with the help of water

Project 2: Task Manager

Project Overview

The Simple Task Manager is a web-based application designed to help users organize their tasks efficiently. It allows users to create, view, update, and delete tasks in a user-friendly interface.

Project Details

The Simple Task Manager is a web-based application enabling users to efficiently manage tasks, featuring user authentication, task creation, filtering, sorting, intuitive interface, and secure data storage with MongoDB.
